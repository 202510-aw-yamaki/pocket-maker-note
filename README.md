# いつも買ってるあのメーカー ポケット帳

## 概要

「いつも買っているメーカー・商品・サイズ」を忘れないための小型メモアプリです。

スーパーやドラッグストアで買い物をしているときに、

* 醤油はどこのメーカーだったか
* ケチャップはカゴメだったか、デルモンテだったか
* にんにくチューブは S\&B だったか、ハウスだったか
* いつも買っているサイズはどれだったか
* 減塩タイプではなく普通タイプだったか
* パッケージの見た目はどれだったか

といった小さな迷いを減らすために使います。

!\[QR Code](QR.png)

## このアプリで解決したいこと

調味料や日用品は、選択肢が少ないからこそ迷いやすいものです。

売り場には似た商品が並んでいて、メーカー名、サイズ、パッケージ、バリエーションが少しずつ違います。

このアプリは、そうした「いつものやつ、どれだっけ？」を買い物中にすぐ確認できるようにするためのポケット帳です。

## 想定する使い方

買い物中にスマホで開き、登録しておいた商品を確認します。

たとえば、醤油を買うときにアプリを開くと、

* アイテム名
* メーカー名
* 商品名やサイズ
* パッケージ写真
* 最後に買った日
* 補足メモ

を確認できます。

これにより、似た商品を間違えて買うことを防ぎます。

## 記録する情報

このアプリでは、以下の情報を記録します。

* アイテム名
* メーカー名
* カテゴリー名
* 商品詳細・型番・サイズ
* 写真
* 最後に買った日
* メモ

## 使用例

### 醤油

* アイテム名: 醤油
* メーカー名: キッコーマン
* 商品詳細: 特選 丸大豆しょうゆ 1L
* メモ: 赤いキャップのやつ。減塩ではない。

### ケチャップ

* アイテム名: ケチャップ
* メーカー名: カゴメ
* 商品詳細: トマトケチャップ 500g
* メモ: デルモンテではなくカゴメ。

### にんにくチューブ

* アイテム名: にんにくチューブ
* メーカー名: S\&B
* 商品詳細: おろし生にんにく 43g
* メモ: 大容量ではなく普通サイズ。

## コンセプト

このアプリは、買い物リストではありません。

買うものを管理するアプリではなく、  
「いつも買っている商品の正解」を忘れないためのメモ帳です。

目的は、買い物中に迷ったとき、

「これで合ってる」

とすぐ確認できることです。
