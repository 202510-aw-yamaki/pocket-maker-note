# Screens

## 1. 商品一覧画面

### 目的

買い物中に、登録した商品をすばやく探す。

### 表示内容

各商品カードに以下を表示する。

- 写真サムネイル
- アイテム名
- メーカー名
- 商品詳細・サイズ
- カテゴリー
- 最後に買った日

### 操作

- 検索
- カテゴリー絞り込み
- 商品詳細へ移動
- 新規追加画面へ移動

### UI方針

- スマホで見やすい1カラム
- カード型UI
- メーカー名を目立たせる
- 写真を左、情報を右に置く
- 検索欄は上部に固定気味で置く

### ワイヤーイメージ

```txt
+--------------------------------+
| いつものメーカー帳             |
| [ 検索する                  ]  |
| [すべて] [調味料] [チューブ]   |
+--------------------------------+
| [写真] 醤油                    |
|        キッコーマン            |
|        特選 丸大豆しょうゆ 1L  |
|        最後に買った日 5/1      |
+--------------------------------+
| [写真] ケチャップ              |
|        カゴメ                  |
|        トマトケチャップ 500g   |
|        最後に買った日 4/20     |
+--------------------------------+
|                         [+追加] |
+--------------------------------+
```

## 2. 商品詳細画面

### 目的

店頭で「これで合っているか」を確認する。

### 表示内容

- 大きな写真
- アイテム名
- メーカー名
- 商品詳細・サイズ
- カテゴリー
- 最後に買った日
- メモ

### 操作

- 編集
- 削除
- 一覧に戻る

### UI方針

- 写真を大きく表示する
- メーカー名を大きく表示する
- 商品詳細を読みやすくする
- ボタンは押しやすくする

### ワイヤーイメージ

```txt
+--------------------------------+
| ← 戻る                         |
+--------------------------------+
|            [ 大きな写真 ]       |
+--------------------------------+
| 醤油                            |
| キッコーマン                    |
| 特選 丸大豆しょうゆ 1L          |
| カテゴリー: 調味料              |
| 最後に買った日: 2026-05-01     |
|                                  |
| メモ:                            |
| 赤いキャップのやつ。減塩ではない。|
+--------------------------------+
| [編集]                  [削除] |
+--------------------------------+
```

## 3. 商品追加・編集画面

### 目的

商品情報を登録・更新する。

### 入力項目

- アイテム名
- メーカー名
- カテゴリー名
- 商品詳細・型番・サイズ
- 写真
- 最後に買った日
- メモ

### 操作

- 保存
- キャンセル
- 写真選択

### UI方針

- 1画面に詰め込みすぎない
- 必須項目をわかりやすくする
- スマホで入力しやすくする
- 写真は1枚だけ

### ワイヤーイメージ

```txt
+--------------------------------+
| ← 戻る                         |
| 商品を追加                      |
+--------------------------------+
| アイテム名                      |
| [ 醤油                       ] |
| メーカー名                      |
| [ キッコーマン               ] |
| カテゴリー                      |
| [ 調味料                     ] |
| 商品詳細・サイズ                |
| [ 特選 丸大豆しょうゆ 1L      ] |
| 写真                            |
| [ 写真を選択 ]                  |
| 最後に買った日                  |
| [ 2026-05-01                ] |
| メモ                            |
| [ 赤いキャップのやつ...       ] |
+--------------------------------+
| [保存する]                      |
+--------------------------------+
```

## 4. 画面遷移

```txt
商品一覧
  ├─ 商品詳細
  │    ├─ 編集
  │    └─ 削除
  └─ 新規追加
```

## 5. レスポンシブ方針

MVPではスマホ優先。

- 360px幅で使えること
- PCでは中央寄せの細いレイアウトでよい
- 複雑な2カラムは不要
